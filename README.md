# DoS

DoS is a C code for to study brute force attacks for the course Pentest Profissional by Desec Security.

## Installation

Download the DoS.c archive.
```bash
wget https://raw.githubusercontent.com/mateusroveda/DoS/master/DoS.c
```
And compile.

```bash
gcc DoS.c -o DoS
```

## Usage

```c
./DoS ip port
```
